
# Star war app
This is a simple react-redux add for demo.
 - Login, search and listing
 - Used APIs from https://swapi.co
 
## Clone repository
You can clone the repo from below link.
```
git clone https://github.com/dsachaniya/xebia-star-war.git 
```
### Project Setup
Clone the project
```
> npm/yarn install
> yarn/yarn start
```
